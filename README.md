# BGM PON SYSTEM

iPad からワンタップで BGM を再生・停止するサンプラーWebアプリ。

---

## 概要

- **再生・操作**：iPad（ローカルLAN経由でブラウザ操作）
- **曲データインポート**：PC（Web UIからアップロード）
- **ボタン設定**：iPad・PCどちらからでも可能

---

## 技術スタック

| レイヤー | 技術 | 理由 |
|---|---|---|
| バックエンド | Node.js / Express | 軽量・ファイル管理・フロントと言語統一 |
| フロントエンド | HTML + Vanilla JS | 依存最小・iPad最適化しやすい |
| タップ最適化 | FastClick | 300ms遅延の除去 |
| 音声再生 | Web Audio API | フェードイン/アウト制御が可能 |
| データ保存 | JSON（サーバーサイド） | 設定の永続化、DB不要 |

## 機能仕様

### ボタン設定

| 項目 | 詳細 |
|---|---|
| 曲アサイン | ボタンごとに音声ファイルを紐付け |
| ラベル | 短いテキストラベルを表示 |
| 未アサイン表示 | 曲がないボタンは明確にDisabled表示 |
| 再生モード | **1btnモード**（長押し中のみ再生） / **2btnモード**（タップで再生・停止トグル） |
| フェードイン | ON/OFF + 秒数設定 |
| フェードアウト | ON/OFF + 秒数設定 |
| 終了タイマー | 秒数設定（再生開始から指定した秒数で停止またはフェードアウト開始） |

### 音声再生

- Web Audio API で再生・停止・フェード制御
- フェード中に別ボタンを押しても正常動作すること
- iPad Safari では初回タップ時に AudioContext を unlock する処理が必須

### ファイルインポート（PCから）

- ブラウザのファイルアップロードUI（multer使用）
- 対応フォーマット：MP3 / WAV / OGG
- アップロード後、ボタン設定画面から即時アサイン可能
